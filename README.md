# Yoyaku

> **Book in Seconds.**

Yoyaku is a smartphone-first reservation platform designed to help users find and book available appointments in seconds.

The first release focuses on massage and wellness businesses. The platform architecture is designed to expand into beauty, healthcare, fitness, hospitality, and any reservation-based industry.

---

# Vision

Booking should be simple.

Users should never need to compare multiple calendars, contact multiple stores, or navigate complicated booking forms.

Yoyaku enables users to search by their desired time, duration, and number of people, and instantly displays only bookable reservation slots.

---

# Mission

Create the world's fastest reservation platform.
