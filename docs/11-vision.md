# 11 Vision

## ビジョン

Yoyakuは、予約をただの入力フォームではなく、店舗と顧客の信頼をつなぐ体験にします。

## なぜYoyakuを作るのか

小規模店舗では、予約受付、電話対応、キャンセル対応、スタッフ調整、売上確認が日々の負担になります。大きなシステムは複雑で、小さな店舗には過剰なこともあります。

Yoyakuは、必要な機能をわかりやすく提供し、店舗が施術や接客に集中できる時間を増やすために作ります。

## 長期構想

長期的には、予約管理だけでなく、店舗運営全体を支える基盤を目指します。

- 予約
- 決済
- 顧客管理
- スタッフ管理
- 売上分析
- 通知
- 多店舗運営
- SaaS課金

## 世界観

Yoyakuが目指す世界は、顧客が迷わず予約でき、店舗が安心して受け入れ準備をできる世界です。予約の不安、確認の手間、無断キャンセルのリスクを減らし、双方にとって心地よい運営を実現します。
