# 01 Project Overview

## 事業概要

Yoyakuは、マッサージ・リラクゼーション・ウェルネス店舗向けの予約管理プラットフォームです。顧客がスマートフォンから空き状況を確認し、予約金を支払って予約を確定できる体験と、店舗側が予約・スタッフ・営業時間・休業日・シフトを管理できる運営基盤を提供します。売上管理や顧客管理は、今後の正式公開に向けて段階的に強化します。

## Yoyakuとは

Yoyakuは、店舗予約に必要な顧客体験と管理業務をひとつにまとめるプロダクトです。

- 顧客向け予約フロー
- 空き時間検索
- 予約金決済
- 管理画面
- スタッフ・シフト・営業時間管理
- 将来的な多店舗管理

## 目的

予約受付の手間、ダブルブッキング、電話対応、手作業の管理業務を減らし、小規模店舗でも使いやすい予約運営基盤を作ることを目的とします。

## 理念

Yoyakuは、店舗と顧客の双方にとって予約をわかりやすく、安心できるものにします。店舗には運営の見通しを、顧客には迷わない予約体験を提供します。

## ターゲット

- 個人経営のマッサージ店
- 小規模リラクゼーションサロン
- 複数スタッフを抱える地域店舗
- 将来的に複数店舗を展開する事業者

## 提供価値

- 顧客が空き枠を即時確認できる
- 予約金により無断キャンセルを抑制できる
- スタッフごとの稼働状況を反映できる
- 店舗側の予約確認・変更・キャンセル管理を集約できる
- 将来的なSaaS展開に耐える運用設計を蓄積できる

## ロードマップ概要

- Phase 1: MVPとして予約・決済・管理画面の基本機能を完成
- Phase 2: β版として実店舗運用、通知、キャンセル管理を強化
- Phase 3: 正式公開としてセキュリティ、監視、サポート体制を整備
- Phase 4: 多店舗対応
- Phase 5: SaaS化
